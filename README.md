# Team Fortress 2 Configuration Vault - an Introduction
This is my personal collection of Team Fortress 2 configuration files which includes, but is not limited to: Class-specific configuration files, custom keybind setups, scripts and anything else that I find interesting. 

Note that this is an evolving setup- not everything is final, but everything is usable.

This repository acts as a modular vault for my setup. Everything from movement scripts to binds and experimental ideas lives here. It is not meant to be perfect, just organized and easy to expand over time.

The structure is built around a central autoexec.cfg file, which serves as the main entry point. Everything is seperated, clean and managed.

The overall philosophy is to keep everything modular, readable, and easy to tweak. Instead of relying on one large configuration file, everything is broken into smaller pieces that can be modified or replaced without breaking the entire setup.

This project is a work in progress. Some scripts may be experimental or unfinished, and changes will happen over time as the configuration evolves.

The setup is designed with flexibility in mind rather than strict competitive optimization. Anyone is free to reuse or adapt parts of it, but it is primarily built for personal use.

To use the configuration, contact me for instructions.

Planned improvements include continuous improvement, testing and improving overall organization and documentation.
